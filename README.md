# Magnate — 3D economy-sim prototype

A self-contained static build of the Magnate visualization (Three.js). All app
JavaScript (deterministic economy core + visualization panels) is inlined into
`index.html`; only the Three.js library loads from a CDN. Open `index.html`
locally, or view the live GitHub Pages site.

Scenes: factory production-line world (streamed), market & finance Sankeys,
trade network, and leaderboard.
